# Security Policy

## Reporting a Vulnerability

We take the security of switchAILocal seriously. If you find a security vulnerability, please do **not** open a public issue. Instead, report it privately.

Please send security reports to: **security@traylinx.com**

In your report, please include:
- A description of the vulnerability.
- Steps to reproduce the issue.
- Potential impact.

We will acknowledge your report and work on a fix as quickly as possible.
