# Cortex Skills Index

This index provides a summary of all specialized skills available to the Cortex Router. Each skill provides a modular set of instructions and tools to enhance the agent's expertise in a specific domain.

---

| Name                      | Description                                                                                                                                                     |
| :------------------------ | :-------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **blog-optimizer**        | Specialized instructions for writing and optimizing Jevvel Labs blog posts. Focuses on SEO, engaging introductions, and the unique 'cyber-brutalist' aesthetic. |
| **frontend-design**       | Create distinctive, production-grade frontend interfaces with high design quality. Use for building web components, pages, artifacts, posters, or applications. |
| **frontend-expert**       | Expert in Modern React, TailwindCSS, and Aesthetic UI Design.                                                                                                   |
| **git-expert**            | Expert in Git workflows, Conventional Commits, and PR management.                                                                                               |
| **go-expert**             | Specialized knowledge for writing professional, idiomatic Go (Golang) code, specifically for the `switchAILocal` codebase.                                      |
| **k8s-expert**            | Expert in Kubernetes, Helm, and Cloud Native deployments.                                                                                                       |
| **mcp-builder**           | Guide for creating high-quality MCP (Model Context Protocol) servers that enable LLMs to interact with external services through tools.                         |
| **skill-creator**         | Guide for creating and updating specialized expertise, workflows, and tools.                                                                                    |
| **switchai-architect**    | Expert on the `switchAILocal` architecture, Go host, and Lua plugin system.                                                                                     |
| **testing-expert**        | Expert in Testing Methodologies (Vitest, Playwright, TDD).                                                                                                      |
| **vision-expert**         | Expert in analyzing images, diagrams, and UI screenshots.                                                                                                       |
| **web-artifacts-builder** | Tools for creating elaborate, multi-component agent HTML artifacts using React, Tailwind CSS, and shadcn/ui.                                                    |
| **webapp-testing**        | Toolkit for interacting with and testing local web applications using Playwright.                                                                               |

---

## Usage

Skills are triggered automatically by the Cortex Router when it detects that the request falls within the skill's domain of expertise. They provide the agent with the procedural knowledge and tools necessary to execute high-quality work in these specialized areas.
